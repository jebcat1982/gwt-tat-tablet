GWT TAT Tablet
====

`GWT TAT Tablet` is a cross-platform HTML5 tablet concept influenced by [TAT Acute prototype](https://www.youtube.com/watch?v=c6rBW0hk8iM) using JavaScript, GWT and HTML5
